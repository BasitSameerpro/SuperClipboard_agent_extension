# Agentic Super Clipboard Extension 🚀

Agentic Super Clipboard is a developer-focused, high-performance browser extension designed to warm, search, and copy a catalog of over 690+ AI system directives, expert prompts, and developer skills directly into your clipboard. It also features a zero-dependency **Folder Bundler** to package local project repositories into single plain-text files.

---

## 📖 How to Use the Extension

### 1. Open the Clipboard Helper
You can open the Agentic Clipboard popup bubble using three quick methods:
* **⌨️ Keyboard Shortcut:** Press **`Ctrl + Shift + Space`** (or `Cmd + Shift + Space` on macOS) to instantly open the native popup anchored to your browser toolbar. Pressing it again when focused closes it.
* **🖱️ Context Menu:** Right-click anywhere on a webpage, text selection, or the extension's toolbar icon, and choose **"Open Agentic Clipboard"**.
* **🌐 Address Bar (Omnibox):** Type **`ac`** in the browser's address bar, press **Space** or **Tab**, and hit **Enter** to open the clipboard utility.

### 2. Search & Copy Commands (Tab 1: Quick Command)
1. Type **`/`** followed by keywords to search through the templates catalog.
2. Use the **Up/Down arrow keys** to navigate the list (the active item automatically scrolls into view).
3. Press **Enter** to select the template.
4. The compiled prompt payload is copied directly to your clipboard, and the popup automatically closes within **50ms** for maximum development speed.

### 3. Browse Prompt Templates (Explore Catalog)
1. Click the **"Explore Catalog"** button inside the popup.
2. Select any of the **6 high-aesthetic Domain Cards** (AI, Software Engineering, Research, DevOps, etc.) to drill down.
3. Filter by subcategory tag pills (e.g. `#frontend`, `#kubernetes`, `#rust`) or type keywords.
4. Click the eye icon (**👁️**) to preview the template payload in detail, or click **"Copy"** to copy it directly.

### 4. Package codebase (Tab 2: Folder Bundler)
1. Select the **"Folder Bundler"** tab.
2. Choose your **Extraction Mode**:
   - *No Compression:* Copies raw file contents.
   - *Minify:* Strips code comments and unnecessary whitespace.
   - *Skeleton:* Generates signature and type summaries only (ideal for long codebases).
3. Click **"Select Folder & Copy Payload"**, pick your project directory, and WXT will package it into your clipboard, ignoring standard build files (`node_modules`, `.git`, etc.) automatically.
